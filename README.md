# 🌟 Influencer Products Website

A modern and responsive **Influencer Products landing page** built using HTML and CSS. This project showcases a clean UI design, product cards, and a featured section tailored for influencer gear and accessories.

---

## 🚀 Live Demo
🔗 https://azizulislamch.github.io/Influencer-Products/

---

## 📂 GitHub Repository
🔗 https://github.com/Azizulislamch/Influencer-Products

---

## ✨ Features

- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎯 Clean and Modern UI
- 🛍️ Product Cards with Ratings and Pricing
- ⭐ Star Rating System with Icons
- 📦 Featured Product Section
- 🎨 Custom Styling with CSS
- 🔘 Interactive Buttons with Icons
- 📐 Flexbox and Grid Layout Usage

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Font Awesome (for icons)

## 📸 Sections Included

- 🔝 Navbar

- 🎯 Featured Product Section  
- 📞 Footer
